# Variables and Data Types in C++

Variables and data types form the foundation of programming in C++. A **variable** is a named storage location in memory used to hold data that your program can manipulate. Each variable has an associated **data type**, which tells the compiler what kind of value it stores and how much memory to allocate.

C++ provides several basic data types such as:

- `int` – for integers  
- `float` and `double` – for decimal numbers  
- `char` – for single characters  
- `bool` – for true/false values  

Understanding how to declare variables, choose appropriate data types, and use them correctly is essential for writing efficient and error-free programs. These concepts will be used in almost every program you write.
