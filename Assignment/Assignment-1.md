# Assignment 1 — Variables and Data Types in C++

Complete the following hands-on exercises to strengthen your understanding of variables and data types in C++. Write and run each program on your system.

---

## Questions

1. **Variable Declaration Practice**  
   Write a C++ program that declares variables of type `int`, `float`, `char`, and `bool`. Assign values to them and print each value with a descriptive message.

2. **User Input and Output**  
   Create a program that asks the user to enter their age (integer) and height (float), then prints them in a formatted sentence.

3. **Type Conversion**  
   Write a program that takes an integer input from the user, converts it to a `double`, and prints both the original and converted values.

4. **Arithmetic with Different Data Types**  
   Declare two integers and one floating-point number. Perform addition, subtraction, multiplication, and division, and print the results.

5. **Character and ASCII Value**  
   Write a program that takes a character as input and prints its ASCII value using type casting.

6. **Boolean Expressions**  
   Create a program that declares two integers and prints the result of at least three boolean comparisons (e.g., greater than, equal to).

7. **Swapping Variables**  
   Write a program to swap the values of two integer variables using a temporary variable and display the result before and after swapping.

8. **Simple Calculator**  
   Build a small calculator that takes two numbers and prints their sum, difference, product, and quotient.

9. **Size of Data Types**  
   Write a program that prints the memory size (in bytes) of `int`, `float`, `double`, `char`, and `bool` using the `sizeof` operator.

10. **Mixed-Type Expression**  
    Create a program that evaluates an expression involving `int`, `float`, and `double` variables. Print the result and explain (in a comment) how type promotion occurs.

---

**Submit all programs with proper comments and meaningful variable names.**
