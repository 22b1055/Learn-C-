# Assignment 2 — Conditional Statements in C++

Complete the following exercises using **only**:

- `if`
- `if-else`
- `else if`

Do **not** use `switch` statements.

Write clean code, use meaningful variable names, and test your programs with different inputs.

---

## Questions

1. Write a program to check whether a number is positive, negative, or zero.

2. Write a program to check whether a number is even or odd.

3. Write a program to find the greater of two numbers.

4. Write a program to find the largest of three numbers.

5. Write a program to check whether a year is a leap year.

6. Write a program to check whether a character entered by the user is a vowel or consonant.

7. Write a program to check whether a character is uppercase, lowercase, or not an alphabet.

8. Write a program to calculate the absolute value of a number without using any built-in function.

9. Write a program to check whether a number is divisible by both 3 and 5.

10. Write a program to check whether a number lies between 10 and 50 (inclusive).

11. Write a program to calculate the percentage of five subjects and print:
   - "Distinction" if ≥ 75  
   - "First Class" if 60–74  
   - "Second Class" if 50–59  
   - "Pass" if 40–49  
   - "Fail" if < 40  

12. Write a program to check whether a number is a multiple of 7 or 11.

13. Write a program that takes a temperature as input and prints:
   - "Hot" if above 35°C  
   - "Warm" if between 20°C and 35°C  
   - "Cold" if below 20°C  

14. Write a program to check whether a triangle is valid given three angles (sum must be 180).

15. Write a program to check whether a number is a three-digit number.

16. Write a program to calculate electricity bill based on units consumed:
   - First 100 units: ₹5 per unit  
   - Next 100 units: ₹7 per unit  
   - Above 200 units: ₹10 per unit  

17. Write a program to check whether a number is divisible by 2, 3, both, or neither.

18. Write a program that takes a number and prints:
   - "Small" if less than 10  
   - "Medium" if between 10 and 100  
   - "Large" if greater than 100  

19. Write a program to determine whether a given number is a two-digit even number.

20. Write a program that checks whether three sides can form a valid triangle (sum of any two sides must be greater than the third).

---

**Submission Instructions:**
- Submit all 20 programs.
- Include comments explaining the logic.
- Test each program with multiple input cases.
